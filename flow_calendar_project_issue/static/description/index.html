Flow Calendar Project Issue

Plan "Project Issues" in the Odoo Calendar

Benefits of planning in the Calendar:
  • Never miss an Issue again. It's planned in the Calendar :)
  • Filter Calendar Events by (your) Issue Stages. E.g. show only Events in stage Analysis and/or Development and/or Testing.
  • Plan an issue right away in the Calendar.
  • See availability of users due to meetings, holidays and all planned project issues.
  • An Issue can be planned as many (linked) Calendar Events.
